# SynQ

> 하나의 회의를 각자의 업무 언어로,  
> **회의 중 이해 차이를 줄이는 AI 서비스**

SynQ는 회의 중 발생하는 이해 차이를 줄이고,  
각자의 역할과 업무 맥락에 맞는 정보를 제공하는 AI 기반 협업 서비스입니다.
